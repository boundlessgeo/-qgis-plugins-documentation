Lessons's documentation
=======================

|current_plugin| is a plugin for QGIS to create and follow step-by-step lessons and tutorials within QGIS.

For users, instruction on how to use the plugin to follow a lesson can be found at :ref:`usage`.

For developers, instructions on how to write new lessons and add them to the plugin can be found at :ref:`writing_lessons`
