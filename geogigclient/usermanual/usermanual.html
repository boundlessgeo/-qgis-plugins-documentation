User Manual¶


GeoGig Quick Start - Installation

This User Manal provides a short introduction to GeoGig and the GeoGig QGIS plugin through simple examples/exercises that demostrate its main features.

Overview¶

GeoGig is a tool for managing change in geospatial datasets with sophisticated workflows and version management. The easiest way to use GeoGig is through the QGIS GeoGig plugin. For more information see the Introduction to GeoGig and GeoGig Quick Start videos.

This Quick Start is designed to get you quickly using GeoGig and the GeoGig-QGIS3 plugin.

Please view the video, above, which walks through this Quick Start.
