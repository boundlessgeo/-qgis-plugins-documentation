GeoGig Client plugin documentation¶

Geogig client is a plugin for QGIS to work with GeoGig repositories using the GeoGig Web API.

GeoGig is an open source tool that draws inspiration from Git, but adapts its core concepts to handle distributed versioning of geospatial data. GeoGig is a Locationtech project, for which Boundless (http://boundlessgeo.com/) contributes actively. For more information about GeoGig see http://geogig.org/
