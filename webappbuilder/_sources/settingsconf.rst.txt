Plugin settings
###############

The plugin can be adjusted using the following settings, to be found in its settings dialog:

* General 
	 - *SDK Service Endpoint*: The URL of the WebSDK server

	 - *Connect authentication endpoint*: The URL of the Connect portal authenitcation service

	 - *Warn when output folder exists*: If output webapp folder exists, ask permission to overwrite it

	 - *Behaviour when opening WAB dialog*: Defines the action to perform when WAB is opened
