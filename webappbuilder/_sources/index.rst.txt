.. _qgis.webappbuilder:

Web App Builder Documentation
=============================

The |current_plugin| is a plugin for QGIS that allows for the easy creation of
web applications based on layers, map compositions and bookmarks, as configured
within a QGIS project. The applications can also include aditional web
services, various controls, and other interactivity.
