Configuration¶

Set API key¶

To use what3words Tools, you will need to have an API key, which can be signed up for here.

Once you have an API key, it should be configured in the plugin using the Plugins ‣ what3words ‣ Set API Key menu item.

Setting up the API key

If you don’t configure the API key, you will be prompted to enter one when calling any of the tools the plugin provides.
