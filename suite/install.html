Installation¶

This section will describe how to install the OpenGeo Explorer plugin.

The OpenGeo Explorer plugin requires QGIS version 2.8 or higher to run, and is installed using the QGIS Plugin Manager.

  1. In QGIS, navigate to Plugins ‣ Manage and Install Plugins.

  2. Click Settings

  3. Scroll down to the bottom and click Add.

  4. Add the following repository to the list of plugin repositories:

    • Name: Boundless plugin repository
    • URL: http://qgis.boundlessgeo.com/plugins.xml

    Adding a new plugin repository

  5. Click OK.

    New plugin repository added successfully

  6. With the Boundless repository added, the OpenGeo Explorer plugin will not be available to be installed. Click Not Installed to see a list of all plugins.

  7. You should see one titled OpenGeo Explorer. Select it and click Install plugin.

    OpenGeo Explorer in the list of plugins

  8. The plugin will be installed. To verify that installation was successful, you should see a new menu in the QGIS menu bar called OpenGeo.

    OpenGeo Explorer menu
